# Installation

Get started with **utills** in seconds. Our package is lightweight, dependency-free, and compatible with all modern JavaScript and TypeScript environments.

## Prerequisites

Ensure you have **Node.js** installed on your machine.

-   **Minimum version:** Node.js v14 or later is recommended.

## Package Manager

Install `utills` as a dependency in your project using your preferred package manager:

### npm

```bash
npm install utills
```
